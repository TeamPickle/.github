# 팀 피클(Team Pickle)
'Pick' for you! 여러분에게 무엇이 좋을지 항상 고민하고 연구하는 팀 피클(Team Pickle)입니다.  

## 팀 피클을 소개합니다!
2020년 2월 작은 챗봇 개발팀으로 시작한 팀 피클은 현재 10여 명이 소속된 팀으로 발전했습니다.<br>
